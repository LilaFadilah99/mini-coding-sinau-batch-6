## ğŸ› Troubleshooting Umum

### Preview Tidak Muncul?

**Problem:** Panel preview kosong atau tidak update
**Solusi:**

1. Pastikan file sudah tersimpan (`Cmd/Ctrl + S`)
2. Refresh preview dengan klik icon refresh di panel preview
3. Kalau masih tidak muncul, buka di browser eksternal (klik kanan â†’ Open in Browser)

---

### AI Tidak Merespon?

**Problem:** Chat AI tidak memberikan jawaban
**Solusi:**

1. Pastikan koneksi internet stabil
2. Coba tutup dan buka kembali AI chat panel
3. Refresh halaman Antigravity
4. Kalau masih bermasalah, tanya pengajar

---

### Kode dari AI Error?

**Problem:** Kode yang diberikan AI tidak bekerja
**Solusi:**

1. **Cek error di browser:**
   - Klik kanan di preview â†’ Inspect â†’ Console
   - Copy error message
2. **Tanya AI untuk debug:**
   - Highlight kode yang error
   - `Cmd/Ctrl + K`
   - Tanya: "Kode ini error: [paste error]. Tolong perbaiki."
3. **Atau tanya pengajar** jika error masih berlanjut

---

### File Tidak Tersimpan?

**Problem:** Perubahan hilang setelah refresh
**Solusi:**

1. Antigravity auto-save, tapi pastikan ada indikator "saved"
2. Manual save dengan `Cmd/Ctrl + S`
3. Check File Explorer - pastikan file ada di lokasi yang benar
